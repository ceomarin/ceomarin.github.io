# ceomarin.github.io

Portfolio personal de QA Automation Engineer construido con Astro y Tailwind CSS v4.

## Stack

- [Astro 6](https://astro.build)
- [Tailwind CSS v4](https://tailwindcss.com)
- [@tailwindcss/vite](https://tailwindcss.com/docs/installation/using-vite)
- [@fontsource/inter](https://fontsource.org)
- [@fontsource/fira-code](https://fontsource.org)

## Secciones

- Hero
- Sobre mí
- Skills (Automatización, Lenguajes, Herramientas)
- Proyectos
- Contacto

## Desarrollo local

```bash
npm install
npm run dev
```

## Deploy

El sitio se despliega automáticamente en [ceomarin.github.io](https://ceomarin.github.io)
cada vez que se hace push a `main` via GitHub Actions.
